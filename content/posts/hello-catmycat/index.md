# 안녕하세요! 🐱

**Cat My Cat** 블로그에 오신 것을 환영합니다!

이 블로그는 Hugo와 PaperMod 테마로 만들어졌으며, Cloudflare Pages에서 호스팅됩니다.

## 블로그 소개

이 블로그에서는 다음과 같은 주제를 다룰 예정입니다:

- 🐱 고양이 이야기
- 💻 웹 개발
- 🚀 Hugo 튜토리얼
- 🛠️ 개발 도구 및 팁

## 기술 스택

이 블로그는 다음 기술로 만들어졌습니다:

- **Hugo** - 초고속 정적 사이트 생성기
- **PaperMod** - 미니멀한 Hugo 테마
- **Cloudflare Pages** - 무료 호스팅
- **Git** - 버전 관리
